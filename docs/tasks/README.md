# Task Templates

Step-by-step guides for common development tasks. Use these as checklists when implementing features.

| [ADD_API_ENDPOINT.md](ADD_API_ENDPOINT.md) | Adding a new REST API endpoint |

## How to Use

1. Pick the template matching your task
2. Follow each step in order
3. Check off items as you complete them
4. Refer to linked documentation for details
